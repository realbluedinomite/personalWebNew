# Personal Website

A modern, responsive personal website with a terminal interface, AI chatbot, and theme switching.

## Features

- **Responsive Design**: Looks great on all devices
- **Dark/Light Theme**: Toggle between light and dark modes
- **Interactive Terminal**: Navigate the website using terminal commands
- **AI Chatbot**: Get information about the website and its owner
- **Multiple Sections**: Home, About, Resume, Portfolio, Blog, and Contact pages
- **Animations**: Smooth transitions and hover effects

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/personal-website.git
   cd personal-website
   ```

2. Open `index.html` in your browser to view the website locally.

## Customization

### Profile Information
- Update your profile picture by replacing `images/profile.jpg`
- Edit the content in `js/main.js` to update your personal information, skills, experience, etc.

### Styling
- The main styles are in `css/styles.css`
- Customize colors, fonts, and layout as needed

### Terminal Commands
Edit the `commands` object in `js/main.js` to add or modify terminal commands.

### Chatbot Responses
Update the `botResponses` object in `js/main.js` to customize the chatbot's responses.

## Deployment

You can deploy this website to any static hosting service like:
- Netlify
- Vercel
- GitHub Pages
- Firebase Hosting

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- Vanilla JavaScript (ES6+)
- Font Awesome for icons
